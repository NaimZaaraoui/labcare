# NexLab Brand Exploration V5

## Objectif

Arreter de chercher un nouveau logo a chaque round et consolider enfin une piste exploitable.

La V5 part de la meilleure intuition recente:

- un symbole fluide
- une vraie lisibilite du nom
- une presence plus calme qu un logo techno futuriste

## Ce Qui A Ete Fige

### Symbole

Le symbole garde l idee du double flux, mais avec une construction plus stable:

- boucle externe plus structurante
- flux interne plus lisible
- point cuivre discret comme accent de controle

Il fonctionne mieux:

- en petit format
- en sidebar
- sur fond clair
- sur fond sombre

### Wordmark

Le nom redevient volontairement sobre.

Le symbole porte la personnalite.
Le mot-symbole porte la confiance.

## Mon Avis

La V5 est la premiere version qui commence a tenir a la fois:

- comme marque
- comme interface
- comme usage reel dans le produit

Elle n est pas parfaite, mais elle est assez stable pour servir de base quasi finale.

## Ajustement Ulterieur

Le principal retour utilisateur sur la V5 etait juste:

- le `N` paraissait trop organique
- les deux couches ne semblaient pas assez alignees

La version integree a l application a donc ete simplifiee ensuite vers une construction plus geometrique:

- meme trajectoire pour les deux couches
- silhouette plus nette
- meilleure sensation d ordre
