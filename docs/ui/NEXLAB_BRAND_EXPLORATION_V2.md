# NexLab Brand Exploration V2

## Pourquoi Le Round 1 N Etait Pas Suffisant

Les premiers logos etaient trop:

- sages
- symetriques
- corporate
- previsibles

Ils pouvaient fonctionner pour un produit propre, mais pas pour une marque memorable.

## Direction Du Round 2

Cette fois, la logique est differente:

- moins de lettre explicite
- plus de symbole-signature
- plus d asymetrie
- plus de tension visuelle
- toujours sobre, mais moins classique

Le but n est pas de dessiner un laboratoire.
Le but est de construire une forme proprietaire qui pourrait devenir `la marque NexLab`.

## Concepts

### 1. Trace Cut

Fichier: `public/branding/nexlab-symbol-trace-cut.svg`

Idee:

- une masse stable
- une incision oblique, nette
- un accent cuivre qui agit comme un point de mesure

Lecture:

- precision
- intervention
- donnee capturee
- systeme qui tranche le bruit

### 2. Split Sample

Fichier: `public/branding/nexlab-symbol-split-sample.svg`

Idee:

- deux volumes decales
- une coupe qui se propage entre eux
- une logique de separation, transformation, analyse

Lecture:

- echantillon
- decomposition
- avant / apres
- laboratoire sans cliche visuel

### 3. Signal Mark

Fichier: `public/branding/nexlab-symbol-signal-mark.svg`

Idee:

- un bloc instrument
- une trajectoire interne qui ressemble a un signal utile
- une sortie cuivre comme point de validation

Lecture:

- lecture analytique
- systeme numerique
- outil de controle
- marque plus technologique

## Mon Avis Honnette

Si tu veux quelque chose de plus unique que le round 1:

- `Trace Cut` est le plus fort comme marque
- `Signal Mark` est le plus credible pour un logiciel LIS
- `Split Sample` est le plus conceptuel

Si NexLab devait avoir un signe qui ne ressemble ni a un microscope ni a un `N` banal, je regarderais en priorite:

1. `Trace Cut`
2. `Signal Mark`
