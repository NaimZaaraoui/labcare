# NexLab Brand Exploration V7

## Nouveau Cap

La V7 part d un constat simple:

- le symbole abstrait n etait pas encore convaincant
- le wordmark simple etait plus juste
- la marque gagnait a etre plus disciplinee

Le choix a donc ete de simplifier franchement.

## Principe

Le coeur de la marque devient:

- `NEX` dans un cadre
- `LAB` ouvert a droite

Pourquoi cela fonctionne mieux:

- lecture immediate
- silhouette stable
- bonne tenue en petit et grand format
- moins d effet gadget

## Ton Visuel

La V7 ne cherche pas a paraitre futuriste.
Elle cherche a paraitre:

- nette
- calme
- serieuse
- assumee

## Composants

La marque a ete adaptee en deux usages:

- un `mark` compact pour les espaces tres contraints
- un `lockup` complet pour login, setup et navigation ouverte

## Integration

La V7 remplace la piste precedente dans l interface:

- navigation laterale
- login
- setup
- changement de mot de passe

## Verdict

Cette version est moins demonstrative, mais plus mature.
Pour NexLab, c est probablement un meilleur choix.
